# Security Policy

## Supported Versions

The following table describes which versions of OwnVideoPlayer are currently supported with security updates.

| Version        | Supported          |
| -------------- | ------------------ |
| Latest release | :white_check_mark: |
| Older releases | :x:                |

Only the latest stable release receives security updates. Users are encouraged to keep their dependency up to date to benefit from the latest fixes and improvements.

## Reporting a Vulnerability

If you discover a security vulnerability in OwnVideoPlayer, please report it responsibly by opening a **private security advisory** on GitHub or by contacting the maintainer directly.

When reporting a vulnerability, please include as much information as possible:

* A detailed description of the issue.
* Steps to reproduce the vulnerability.
* The affected version.
* Any proof of concept or logs that help reproduce the problem.

After receiving your report:

* You will receive an acknowledgment within **5 business days**.
* The vulnerability will be investigated and validated.
* If confirmed, a fix will be developed and released as soon as reasonably possible.
* You will be notified of the outcome once the issue has been resolved or if it is determined not to be a security vulnerability.

Please **do not disclose security vulnerabilities publicly** until a fix has been released, allowing users time to update safely.

Thank you for helping keep OwnVideoPlayer secure.
